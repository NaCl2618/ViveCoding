# A4 단월 달력 템플릿 — 사용설명서 📋

**간단 소개**
- 이 프로젝트는 A4 한 페이지 인쇄용 단월 달력 템플릿입니다. 브라우저에서 열어 연도·월·방향을 선택한 뒤 인쇄하면 깔끔한 A4 달력을 얻을 수 있습니다.

## 사용 방법 🔧
1. `Calender/Calender.html` 파일을 브라우저로 엽니다.
2. 툴바에서 다음 항목을 설정합니다:
   - **연도** (`#year`) — 연도 입력
   - **월** (`#month`) — 월 선택
   - **방향** (`#orientation`) — `가로 (A4)` 또는 `세로 (A4)`
   - **주 시작 요일** (`#startMonday`) — `월요일 시작` 또는 `일요일 시작`
   - **테두리 두께** (`#borderWidth`) — mm 단위 (0.2 ~ 1.5)
   - **공휴일/중요일** (`#holidays`) — `YYYY-MM-DD:이름` 형식의 쉼표 구분 목록
3. **적용 / 새로고침** 버튼을 누르거나 입력을 변경하면 달력이 갱신됩니다.
4. 인쇄: 브라우저의 인쇄 대화상자를 열고 **A4**, **여백 없음**, **배경 그래픽 인쇄**, **헤더/푸터 끄기**를 권장합니다.

## 공휴일 입력 예시 🗓️
```text
2025-01-01:신정, 2025-02-14:발렌타인데이, 2025-12-25:크리스마스
```
- 위 형식으로 입력하면 해당 날짜에 태그가 표시되고 날짜가 강조됩니다.

## 팁 & 권장 설정 💡
- 인쇄 호환성: 브라우저별로 인쇄 결과가 다를 수 있으므로, 원하는 출력이 나올 때까지 인쇄 미리보기를 확인하세요.
- 접근성: 키보드로 입력 가능한지, 화면 판독기에서 의미가 잘 전달되는지 점검하는 것이 좋습니다.
- 확장 기능 아이디어: 공휴일 JSON/ICS 임포트, PDF 저장 버튼, 공휴일 유효성 검사 및 오류 피드백 추가.

## 개발자 노트 🔧
- 파일: `Calender/Calender.html` — 인라인 CSS + 자바스크립트로 동작합니다.
- 주요 ID: `#year`, `#month`, `#orientation`, `#startMonday`, `#borderWidth`, `#holidays`, `#apply`, `#page`, `#grid` 등.
- 브라우저: 최신 Chromium 계열(Chrome, Edge)에서 테스트 권장. 인쇄 동작은 브라우저마다 다를 수 있습니다.
